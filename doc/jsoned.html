json_encode

(PHP 5 >= 5.2.0, PHP 7, PECL json >= 1.2.0)

Returns a string containing the JSON representation of the supplied value. The encoding is affected by the supplied options and additionally the encoding of float values depends on the value of serialize_precision.

Parameters value The value being encoded. Can be any type except a resource. All string data must be UTF-8 encoded.
